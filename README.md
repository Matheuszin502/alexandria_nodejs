## 📖 Descrição
Este projeto se trata de uma livraria virtual, possuindo tanto front-end quanto back-end, o intuito do projeto é praticar
habilidades de desenvolvimento fullstack.

### 💻 Tecnologias
- Javascript (Express.js)
- MySQL
- HTML
- CSS

### 👀 Como visualizar a livraria
Basta clonar o projeto em uma pasta, abrir a pasta no VSCode e rodar no terminal o comando npm start e então acessar o
endereço localhost:3000.
